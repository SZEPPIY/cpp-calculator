Simple C++ Calculator
This is my very first programming project. I built this clean, console-based calculator during my first year as a Computer Science Engineering student.

How it works:

It asks for two numbers and an operator (+, -, *, /).

It calculates and displays the result.

It has basic error handling to prevent division by zero.

It allows you to run multiple calculations in a loop.

Technologies:

Language: C++

IDE: Microsoft Visual Studio
